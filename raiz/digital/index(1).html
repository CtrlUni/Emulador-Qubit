O que é DFT?

DFT significa Transformada Discreta de Fourier e é usado para dividir uma onda em múltiplas ondas de frequências diferentes. Embora as transformadas de Fourier sejam tecnicamente calculadas usando integração, a DFT usa soma para que possa ser calculada (infelizmente, os computadores não podem contar até o infinito, o que é necessário para a integração). A fórmula é mostrada à direitaabaixo e requer conhecimento do número complexo sistema a ser usado.

Como o DFT é utilizado aqui?

Existem duas maneiras pelas quais o DFT pode ser utilizado neste programa.

O algoritmo ilustrado à esquerdaacima Esta é a principal função deste aplicativo. Ele recebe uma lista de coordenadas como entrada e a executa através do DFT algoritmo e gera informações para desenhar uma infinidade de ondas (como círculos).

A segunda maneira (atualmente não implementada) é apenas passar uma série de coordenadas y, podendo recriar qualquer onda como uma infinidade de ondas com frequências diferentes (mais uma vez, representadas como círculos), assumindo que os pontos estão se movendo distância (como se o eixo x fosse o tempo).
